# QS World University Rankings Data

This folder is ready for your QS World University Rankings CSV files.

## Quick Start

Simply drop your CSV files here with these names:
- `qs-rankings-2024.csv`
- `qs-rankings-2023.csv`
- `qs-rankings-2022.csv`

That's it! The app will automatically load and use your data.

## CSV Format

Your CSV files should have headers in the first row. Common column names:
- rank, university_name, country, city, overall_score
- Any additional columns from QS rankings are welcome

## Example CSV Structure:
```csv
rank,university_name,country,city,overall_score
1,Massachusetts Institute of Technology,United States,Cambridge,100
2,University of Cambridge,United Kingdom,Cambridge,99.2
3,University of Oxford,United Kingdom,Oxford,98.9
```

## Data Source

Download official QS World University Rankings from:
- https://www.topuniversities.com/world-university-rankings

## Notes

- Keep the file names exactly as shown above
- Use UTF-8 encoding
- First row should be column headers
- The app will handle the rest automatically
